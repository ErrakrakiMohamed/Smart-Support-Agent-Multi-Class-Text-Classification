# Smart Support Agent

## Overview
The "Smart Support Agent" is a multi-class text classification project designed to route customer support tickets to the appropriate department (e.g., Billing, Technical Issue, Shipping, General Inquiry).

## Features
- **Data Pipeline**: Reproducible steps for data ingestion and preprocessing.
- **Model Registry**: Model versioning and tracking.
- **CI/CD**: Automated testing using GitHub Actions.
- **Deployment**: Streamlit app deployed on Hugging Face Spaces.

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the app:
   ```bash
   streamlit run app.py
   ```
